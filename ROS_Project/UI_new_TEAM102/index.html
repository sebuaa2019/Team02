We give our customer the best robot to build Smart Home

This project is a software engineering course required by the School of Computer Science, Beijing University of Aeronautics and Astronautics in the spring of 2019. Based on the software and hardware resources provided by the course, this project develops a robot which can be used for sorting indoor scenes or warehouses such as home or office, automatically or manually building maps according to sensor data, and realizes path planning, motion and updating map data on this basis, as well as using robotic arms to capture objects according to image information. This project can be used as a prototype to try out the basic functions of Indoor Intelligent robots, and take this opportunity to learn to cooperate in engineering development and learning the relevant knowledge of embedded systems.
本项目是北京航空航天大学计算机学院2019年春季学期所开设的《软件工程》课程要求开发的项目。 本项目在课程所提供的软硬件资源的基础上，开发一款使用于如家中或办公室等室内场景或仓库分拣，根据传感器数据自动或手动进行地图建立， 并在此基础之上实现路径规划、运动与更新地图数据，以及根据图像信息利用机械臂完成物体抓取的机器人。 本项目可以作为尝试室内智能机器人的基本功能的原型机，并可以借此机会，学习合作进行工程化开发和学习嵌入式系统的相关知识。

核心技术之目标物检测抓取

物品检测识别主要是通过Kinect2模块和PCL的平面检测算法实现。 Kinect2模块使用Kinect2视觉传感器获取外界情形信息并以三维点云的形式存储，分析模块使用PCL的平面检测算法对三维点云进行分析， 得到独立点云团簇，每个点云团簇视为一个物品。
考虑到特定情况下，物品的空间位置信息可以通过其他形式补全，因此还引入了更为简洁的、二维层次上的图像识别技术来对前述三维检测算法形成补充。

核心技术之路径规划

Navigation模块主要包含定位和导航两个方面的功能。 定位功能通过蒙特卡洛自适应定位算法实现，使用概率理论对机器人位置进行估计； 导航功能通过move_base包实现，这个包将机器人导航需要用到的地图、坐标、路径和行为规划器连接到了一起， 同时还提供了导航参数的设置接口。
